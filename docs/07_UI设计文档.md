# UI 设计文档

**版本：** v1.0
**更新日期：** 2026-05-22

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **健康感与食欲感并存** | 绿色主调传达健康，暖橙点缀激活食欲，避免医疗感的冷蓝白 |
| **信息置信度可视化** | 饮食标签需区分"已确认"与"推断"，过敏原警告必须显著区分于普通信息 |
| **搜索为核** | 搜索框在所有页面首屏内可见，Facet 面板始终可触达，不因内容多而被推出视口 |
| **渐进式披露** | 相关性解释、营养详情折叠隐藏，用户主动点击才展开，不干扰默认浏览体验 |
| **可访问性优先** | 过敏原警告不依赖颜色区分（颜色盲用户），同时使用图标+文字+颜色三重编码 |

## 2. 色彩系统

| 角色 | Hex | RGB | 用途 |

| Primary Green Light | `#4DB87A` | rgb(77, 184, 122) | Hover 状态、进度条 |
| Primary Green Dark | `#1E7A42` | rgb(30, 122, 66) | Pressed 状态 |

| 角色 | Hex | 用途 |
|------|-----|------|

| 角色 | Hex | 用途 |

## 3. 字体系统

### 3.1 字体族

- **主字体**：`"PingFang SC", "Microsoft YaHei", "Helvetica Neue", Arial, sans-serif`
- **数字/代码**：`"SF Mono", "Fira Code", monospace`（营养数据展示用）

### 3.2 字号规范

| 场景 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| Hero 标题 | 36px | 700 | 1.3 | 首页主标语 |
| 页面大标题 | 24px | 700 | 1.4 | 页面 H1 |
| 卡片标题 | 18px | 600 | 1.4 | 餐厅名称 |
| 正文 | 14px | 400 | 1.6 | 说明文字、描述 |
| 小标签 | 12px | 400 | 1.5 | 饮食标签徽章文字 |
| 极小注释 | 11px | 400 | 1.4 | 数据来源说明 |
| 营养数字 | 16px | 600 | 1.2 | 热量/蛋白质等数值（等宽字体） |

## 5. 组件规范

### 5.1 搜索框

- 高度：48px（首页 Hero 区）/ 40px（Header 内嵌）
- 圆角：24px（胶囊形）
- 边框：`1px solid #E8E0D5`，focus 时 `2px solid #2D9B5A`
- 内嵌搜索图标（左侧，16px，`#AAB4B4`）
- Autocomplete 下拉：`border-radius: 12px`，`box-shadow: 0 8px 24px rgba(30,42,42,0.12)`

## 6. 图标系统

- **风格**：Outline（线框）为主，24px 基础尺寸
- **主图标库**：Ant Design Icons（已包含在 Ant Design 5.x 中）
- **饮食类型图标**：自定义 SVG（由 frontend-illustration-generator skill 生成），遵循本文档色彩规范
- **描边**：2px，`stroke-linecap: round`，`stroke-linejoin: round`

## 7. 空状态设计

| 场景 | 图示元素 | 提示文案 |
|------|---------|---------|
| 无搜索结果 | 放大镜 + 飘落的蔬菜叶子 | "没有找到符合条件的餐厅，试试减少过滤条件？" |
| 无收藏 | 空书签 + 虚线圆圈 | "还没有收藏，搜索并收藏喜欢的餐厅吧" |
| 今日无饮食记录 | 空碗 + 筷子 | "今天还没有记录，点击"+"开始记录饮食" |
| 未设置饮食档案 | 人物轮廓 + 问号 | "设置饮食档案，获得更精准的搜索结果" |

空状态图使用 `empty-state` 尺寸预设（320×240），颜色使用淡化的品牌色，背景 `#F7F3EE`。

## 8. 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|---------|
| desktop (xl) | ≥ 1200px | Facet 面板 240px 固定左栏 + 右侧结果区 |
| laptop (lg) | 992-1199px | Facet 面板收起为顶部筛选栏 |
| tablet (md) | 768-991px | 单列布局，顶部筛选折叠 |
| mobile (sm) | < 768px | v1.0 不做移动端优化，可横向滚动 |

## 9. 可访问性规范

- 所有交互元素最小点击区域：44×44px
- 颜色对比度：正文文字 `#1E2A2A` on `#F7F3EE` = 10.2:1（AAA）
- 过敏原警告：颜色 + ⚠️图标 + `aria-role="alert"` 三重编码，色盲用户亦可识别
- 搜索结果卡片：`<article>` 语义标签，`aria-label="餐厅名称"`
- 饮食标签：`title` 属性提供完整描述（如 `title="纯素食，不含任何动物产品"`）
